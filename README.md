# 📂 Estudos de PHP - Do Básico ao Processamento de Dados

Este repositório documenta minha jornada de aprendizado em **PHP**, partindo dos conceitos fundamentais de sintaxe até o desenvolvimento de lógica para processamento de dados e integração com formulários.

> 🎯 **Objetivo Profissional:** Estudante de **Análise e Desenvolvimento de Sistemas (ADS)** com foco em migrar para a área de TI, construindo uma base sólida em desenvolvimento back-end.

## 🚀 Evolução Técnica e Conceitos Explorados

Ao longo destes exercícios, foram aplicados os seguintes conhecimentos técnicos:

* **Sintaxe e Output:** Uso de `echo` para renderizar conteúdos dinâmicos e integração com tags HTML (`<h1>`, `<p>`, `<mark>`).
* **Comentários e Documentação:** Prática de boas práticas com comentários de linha (`//`, `#`) e de bloco (`/* */`).
* **Manipulação de Variáveis:** Declaração e exibição de tipos variados (Strings, Integers) e concatenação.
* **Estruturas de Controle e Lógica:**
  * Condicionais dinâmicas com `if`, `elseif` e `else`.
  * Estruturas de repetição `for` e `foreach`.
  * Controle de fluxo com o comando `break`.
* **Processamento de Dados:**
  * Manipulação de Arrays unidimensionais e multidimensionais (listas de funcionários).
  * Recebimento de dados via método `$_POST` para interação com formulários.
* **Estruturação de Dados com HTML:** Criação de tabelas com cabeçalhos horizontais (`<th>`) e verticais.
